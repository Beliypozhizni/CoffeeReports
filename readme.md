# Coffee Reports

CLI-утилита для анализа потребления кофе студентами из CSV-файлов и генерации отчётов.

---

## Установка зависимостей

В проекте для управления зависимостями используется Poetry.

Следует создать виртуальную среду и активировать её (Windows):

```
python -m venv .venv
.\.venv\Scripts\activate
```

Установить зависимости:

```
poetry install
```

## Использование

Пример:

```
python -m coffee_reports --files math.csv physics.csv programming.csv --report median-coffee
```

## Тесты

Зависимости для тестов находятся в группе "dev"

Запуск тестов с покрытием:

```
pytest
```

## Добавление новых отчетов

В <i><b>coffee_reports/reports</b></i> следует создать новый .py файл, 
в котором нужно реализовать класс, унаследованный от <i><b>ReportBuilder</b></i>.

Затем нужно добавить экземпляр данного класса в словарь <i><b>\_REPORTS</b></i>,
находящемся в <i><b>coffee_reports/reports/\_\_init__</b> по аналогии с существующим отчетом.
